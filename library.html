Neumorphism

Accessibility Mode:

Library Download

Download

It's that easy!

Just click the button above to download the compressed version of the library. So far it is in development, so the code is a mess and interaction is not exactly smooth. But by the time the website is polished it will be properly usable. Still, if you are interested, you technically can install it.

Just unpack the folder and move it to your project directory. To use its classes, you will need to import the main file, that is "[folder-directory]/Neumorphism/Neumorphism.css", as a regular stylesheet.

If you wish to tweak values such as the background color, you can change them in the Values.CSS file in the folder.

That's it! Use it to your heart's content. Just remember that everything is subject to change.
