Reflection

What have I learned in developing this website

When starting the assignment, I already had some experience with the main components of web development - various HTML components and rules, CSS, general styling conventions, and JavaScript fundamentals. Despite that, it was a great learning experience in multiple ways.



First of all, it gave me an opportunity to hone my skills in the aforementioned areas of web design. At the start of the course, I was rather slow and unconfident. It took me roughly a dozen hours to get the first of now twenty CSS files that comprise the styling of this project. Granted, it contained the fundamental building blocks of the whole design, but it was still comparatively small. But after that I was way more familiar with CSS, so all of the remaining files took about the same amount of time.

Besides just plain practice, I also expanded my knowledge of niche HTML and CSS features, such as variables, transitions, and multiple spreadsheet import for the latter. I was also missing some of the basic HTML knowledge that never came up in my previous studies. For example, I never dealt with forms before. This projct prompted me to do more research and get familiar with all of the above.

Finally, I have learned quite a bit about the main principles of web design. A lot of these things seem like common sense, but you don't realise how much you've been doing wrong until you think reflect on it. For example, the idea that large images need to be resized and originals be available on clicking the smaller image. It seems painfully obvious when you think about it, but it was a new discovery for me.

Overall, it was a great learning experience and I quite fond of what came out of it. I wanted to explore Neumorphism for a while, and this project allowed me to do just that. The result is this very website, and I really hope that you enjoyed visiting it as much as I did making it.
