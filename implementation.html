Implementation

Before we begin

This page explores the details of CSS behind Neumorphism. However, if you wish to incorporate it into your website without re-writing the entire thing from scratch, you can simply download a finished library from the Library Download Page.



Initial setup

As you have probably noticed, everything I mentioned about Neumorphism so far was regarding the elements on the page. However, before we can get to that there are some fundamental things that need addressing.

1

Background color. Typically it has to be very light. However, it is important to remember that for highlights to be noticeable the background should be slightly dimmer than usual. For example, 235/255 for a not-quite-white page.

2

Font. The typical Times New Roman rarely fits any page, let alone a futuristic one. The choice of the font family is yours. I find that monospace fonts generally look good, because they appear less dense. Alternatively, increasing the letter spacing by a miniscule amount prevents clutter. 


html {
    background-color: rgb(235, 235, 235);
    font-family: Rockwell;
    letter-spacing: 0.5px;
}

I am a good-looking text



Main elements

Finally it is time to move on to the 3D effect that defines the style. Note that this is an example implementation and yours is almost guaranteed to have some slight alterations.
An outset element can be defined with just 4 CSS rules.

1

Background and Border. Because the element should look like a part of the page, its background is set to the color of the page.

As for the border, it has to be rounded. Square divs don't fit for various reasons. It is up to you how big you want the curvature - anything from a few pixels all the way to 50% (completely round) works. I personally settled for 10px on normal elements and 50% for some of the buttons.


W3Schools on background-color
W3Schools on border-radius

2

Shadow. A simple shadow for an element. It is created using a suitably named box-shadow property, that creates a shadow with a specified blur, offset, and color. Just choose where your imaginary light source is, usually it's top-left, and offset the shadow in the opposite direction. The exact color of the shadow depends on your choice of color palette. One thing to note is that it's best to give it a little bit of transparency using rgba colors.


W3Schools on box-shadow
W3Schools on RGBA

3

Highlight. Finally, the most important detail - the highlight. It is created using the same box-shadow property as the shadow, despite the name. It should be offset towards the light, opposite of the shadow, and should have a really bright color. If the background color you chose is light, the highlight will probably have to be full-white to be visible. Just play around with the values until it feels right.


W3Schools on box-shadow
W3Schools on RGBA 

div {
    background-color: rgb(235, 235, 235);
    border-radius: 10px;
    box-shadow: 2px  2px 4px rgba(0,0,0, 0.5),
               -2px -2px 4px white;
}
div.

With that we have created the main design of the style. Applying it to paragraphs, headers, and links will already give you an interesting look. But there is so much to Neumorphism than a single design!



Buttons and Insets

The next step in advancing your design would be to create the opposite of an outset element - an inset. It would go into the page rather than out of it. It is really simple to create, but gives way to much more intricate designs.

1

Inset Property. Ironically, the entire inset element can be created from what we already established by adding just one word to the box-shadows - inset. By adding it to each of the shadows and also flipping the colors, we get the desired effect. It is usually added to things such as input boxes or anything clickable, which we'll see right now.

2

Making a button. A simple button is styled like a regular outset element. Maybe you would want to make it round, but the box-shadows stay the same. However, when the button is clicked, it should become inset. You can either completely inset using the technique given in the previous paragraph, or you can do something even better - add a slight inset on an outset button, by combining all of the box-shadow properties together and adding a thin border.
Finally, add a bit of a glow to the text when the button is pressed. You can use text-shadow for that. Doesn't that just look perfect?


W3Schools on border
W3Schools on text-shadow 
button {
    border-radius: 10px
    box-shadow: 2px  2px 4px rgba(0,0,0, 0.3),
               -2px -2px 4px white;
}

/* Inset on press */
button:focus {
    box-shadow: 2px  2px 4px rgba(0,0,0, 0.5),
               -2px -2px 4px white,
                2px  2px 4px rgba(0,0,0, 0.2) inset,
               -2px -2px 4px white            inset;
    color: aqua;
    text-shadow: 0 0 3px rgb(0, 150, 255);
}
inset div.
